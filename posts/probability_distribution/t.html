t分布の性質

Takeda Katsuji2025/10/03約7分統計数理確率分布t分布ヤコビアン確率密度関数連続型ベータ関数ガンマ関数

t分布の性質

t分布は 分散が未知の場合の仮説検定でよく使用される分布 です.

t分布もしくは スチューデントのt分布 と言われます.

定義にある標準正規分布とカイ二乗分布との関係は必ず覚えた方がいいでしょう. (統計検定には頻出！)

定義

Y∼N(0,1), X∼χ2(n), かつ Y⊥X (独立)のとき,

T=YX/n∼t(n)

は自由度 n の t 分布に従う.
ただし, Y は標準正規分布, X は自由度 n のカイ二乗分布を表す.

1. 確率密度関数

確率密度関数 (PDF)

自由度 n の t 分布 T∼t(n) の確率密度関数は

f(t)=Γ(n+12)nπΓ(n2)(1+t2n)−n+12

また n=1　のとき,

f(t)=Γ(1)πΓ(12)(1+t2)−1=1π(1+t2)

これを特別に(標準) コーシー分布 という.

確率密度関数のグラフ

解説

標準正規分布と非常によく似た形をしていて、自由度(degrees of freedom)が増えるにつれて

標準正規分布と一致 するようすが確認できます.

特徴として、自由度が小さい場合, 標準正規分布より 山が低く、裾が厚い です.

t分布の確率密度関数の導出は

  • 変数変換
  • ヤコビアンの計算
  • ガンマ関数の定義

などなど為になる計算が豊富です.ぜひ理解してできるようにしてください.

おすすめ

導出には標準正規分布とカイ二乗分布を使用します.二つの分布についてはこちらをご覧ください.

導出の手順はこちら

導出

Y∼N(0,1), X∼χ2(n), かつ Y⊥X (独立)のとき,
それぞれの周辺確率密度関数を fX(x),fY(y)とする.

fX(x)=12n/2Γ(n2)xn2−1e−x2fY(y)=12πe−12y2

T=YX/n, U=X で変数変換.

  • 逆変換
    x=u, y=unt

  • ヤコビアン |J|

J=(∂x∂t∂x∂u∂y∂t∂y∂u01unt2un)=(01unt2un)|J|=|0⋅t2un−1⋅un|=un

したがって U,T の同時分布 f(u,t) は,

f(u,t)=f(x,y)|J|=fX(x)fY(y)un=fX(u)fY(unt)un=12n/2Γ(n2)un2−1e−u212πe−u2nt2un=12nπ2n/2Γ(n2)un−12e−12u(1+t2n)

ここで周辺化して T の確率密度関数 fT(t) を求めると,

fT(t)=∫0∞f(u,t)du=12nπ2n/2Γ(n2)∫0∞un−12e−12u(1+t2n)du

ここで指数部分を文字で置いて見通しを良くしましょう.

  • a=n−12,b=12(1+t2n)
fT(t)=12nπ2n/2Γ(n2)∫0∞uae−budu

この式の積分部分を I とすると

I=∫0∞uae−budu

z=bu で置換積分すると

I=∫0∞(zb)ae−z1bdz=1ba+1∫0∞zae−zdz=1ba+1∫0∞z(a+1)−1e−zdz

ガンマ関数 の定義より,

I=1ba+1Γ(a+1)=1ba+1Γ(n+12)

さて fT(t) は,

fT(t)=12nπ2n/2Γ(n2)1ba+1Γ(n+12)=1nπ2n+12Γ(n2)1bn+12Γ(n+12)=1nπΓ(n2)Γ(n+12)(2b)−n+12=Γ(n+12)nπΓ(n2)(1+t2n)−n+12

導出終了.

2. 累積分布関数

累積分布関数 (CDF)

自由度 n の t 分布 T∼t(n) の累積分布関数は
初等関数での閉じた形はなく，正則不完全ベータ関数 I で表す
(統計検定の範囲外なので紹介のみ)

ただし x(t)=nt2+n

3. 期待値

期待値 (Mean)

t 分布 T∼t(n) の期待値は

E[T]={0,n>1,未定義,0<n≤1.

（対称性により存在すれば 0, 存在条件は n>1）

興味深いのは 0<n≤1 つまり コーシー分布のときでも期待値が定義されません.

導出の手順はこちら

導出

PDF を

f(t)=C(1+t2n)−n+12,C=Γ(n+12)nπΓ(n2)

とする.

対称性 f(t)=f(−t) より，期待値が存在すれば E[T]=0.

4. 分散

分散

t 分布 T∼t(n) の分散は

V[T]={nn−2,n>2,未定義（発散）,1<n≤2,未定義,0<n≤1.

（n≤1 は平均が未定義. 1<n≤2 は平均は 0 だが二乗積分が発散）

導出の手順はこちら

導出

PDF を

f(t)=C(1+t2n)−n+12,C=Γ(n+12)nπΓ(n2)

とする. 対称性より n>1 で E[X]=0 なので V(X)=E[X2] を計算する.

E[T2]=2C∫0∞t2(1+t2n)−n+12dt.

u=t2/nで置換すると,

  • 逆変換 t=nu
  • 微分 dt=n2u−1/2du
  • 範囲 t:0→∞,u:0→∞
E[T2]=Cn3/2∫0∞u1/2(1+u)−n+12du

ここで z=u1+u とおくと

  • 逆変換 u=z1−z
  • 微分 du=1(1−z)2dz
  • 範囲 u:0→∞,z:0→1
E[T2]=Cn3/2∫01(z1−z)12(11−z)−n+121(1−z)2dz=Cn3/2∫01z12(1−z)n−42dz=Cn3/2∫01z32−1(1−z)n−22−1dz=Cn3/2B(32,n−22)

ただし B はベータ関数.

ここからベータ関数とガンマ関数の性質を使用するときれいに約分できます.

  • B(x,y)=Γ(x)Γ(y)Γ(x+y)
  • Γ(x+1)=xΓ(x)

さて C も代入して丁寧に整理していきましょう.

E[T2]=Γ(n+12)nπΓ(n2)nnB(32,n−22)=nπΓ(n+12)Γ(n2)B(32,n−22)=nπΓ(n+12)Γ(n2)Γ(32)Γ(n−22)Γ(n+12)=nπ⋅Γ(32)Γ(n−22)Γ(n2)=nπ⋅π2Γ(n−22)Γ(n2)=n2⋅Γ(n−22)Γ(n2)=n2⋅Γ(n−22)Γ(n2−1+1)=n2⋅Γ(n−22)n−22Γ(n−22)=n2⋅1n−22=nn−2.

したがって,

E[T2]=nn−2(n>2).

ゆえに n>2 で V[T]=nn−2. 一方 1<n≤2 はベータ積分が発散し分散は未定義. n≤1 は平均も未定義.

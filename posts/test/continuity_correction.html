半整数補正

Takeda Katsuji2025/09/29約2分統計検定1級統計数理検定検定正規近似連続修正半整数補正

半整数補正

半整数補正とは二項分布などの離散型分布を正規近似で連続型の分布に近似するときに近似精度を上げる方法です.
この記事ではグラフやアニメーションとともに、この近似がどの程度有効か議論していきたいと思います.

解説

二項分布 X∼Bin(n,p) の正規近似は

X∼N(np,np(1−p))

今回 Pr(X≤c) の値を

  • 正確確率(Exact) オレンジ色の長方形の面積の和
  • 正規近似・半整数補正なし(Non CC) 赤色の線より左側の曲線下面積
  • 正規近似・半整数補正あり(CC) 緑色の線より左側の曲線下面積

この3種類値を比較していきます.

3種類の面積のグラフ

Exactとの差のアニメーション1

設定

二項分布の 確率 p を固定して n を増加 してExactとの差分をプロットします。

  • 分布: X∼Bin(n,p)（p=0.5）
  • 片側: 左片側（Pr(X≤c)）
  • しきい値: c=⌊an⌋（本文では a=0.35）
  • n の範囲: nmin=10, nmax=100

解説

標準正規分布の分布関数を Φ とし，μ=np, σ=np(1−p) とする.

  • Exact のとき
Pr(X≤c)=∑k=0c(nk)pk(1−p)n−k.
  • 補正なしのとき
Pr(X≤c)NonCC≈Φ(c−μσ).
  • 半整数補正ありのとき
Pr(X≤c)CC≈Φ(c+12−μσ).

差の可視化は

ΔNon CC(n)=Φ(c−μσ)−Pr(X≤c)ΔCC(n)=Φ(c+12−μσ)−Pr(X≤c).

まとめ

左片側では小さな n で ΔNon CC<0（過少推定）になりやすい傾向にあります.
一方で 半整数補正は小さな n でも差を大きく縮めている ことがわかります.

Exactとの差のアニメーション2

設定

二項分布の 確率 n を固定して p を増加 してExactとの差分をプロットします。

  • 分布: X∼Bin(n,p) (n=50)
  • 片側: 左片側（Pr(X≤c)）
  • しきい値: c=⌊an⌋（本文では a=0.35）
  • p の範囲: pmin=0, pmax=1

まとめ

  • 本稿では n=50 を固定し，p を [0,1] で動かして左片側 Pr(X≤c) の近似誤差を比較しました．
  • 近似なし（Non CC）は，μ=np が c+12 に近づく付近で誤差が最大化しやすいことを確認しました．
  • 半整数補正（CC）は，この最大誤差点付近でも誤差を大きく抑えられることを示しました．
  • 例として c=2 の設定では，誤差は p=(c+12)/n=0.05 付近で最大化しやすく，CC による改善が明瞭に観察できます．
